Macros

As a warrior, it's really important to start most of your abilities with /startattack.
However, /startattack is placed after Heroic Strike/Cleave so that it gets applied before your attack goes off.


Sunder Armor Macro
          
#showtooltip
/startattack
/cast Sunder Armor

Hamstring Macro
          
#showtooltip
/startattack
/cast Hamstring

Revenge Macro
          
#showtooltip
/startattack
/cast Defensive Stance
/cast Revenge

Whirlwind Macro
          
#showtooltip
/startattack
/cast Berserker Stance
/cast Whirlwind

Bloodthirst Macro
          
#showtooltip Bloodthirst
/startattack
/cast Bloodthirst
/cancelaura Bloodthirst

Heroic Strike Macro
          
#showtooltip
/cast Heroic Strike
/startattack

Cleave Macro
          
#showtooltip
/cast Cleave
/startattack

Execute Macro
          
#showtooltip Execute
/startattack
/cast Execute

Berserker Rage Macro
          
#showtooltip
/startattack
/cast Berserker Stance
/cast Berserker Rage

Pummel Macro
          
#showtooltip Pummel
/startattack
/cast [form:2,worn:Shields] Shield Bash
/cast [noworn:Shields] Berserker Stance
/cast [form:3] Pummel

Smart Pummel (mouseover) Macro
          
#showtooltip Pummel
/startattack
/cast [form:1/2,worn:Shields,@mouseover,harm,nodead][form:1/2,worn:Shields] Shield Bash
/stopcasting [form:1/2,noworn:Shields]
/cast [noworn:Shields] Berserker Stance
/cast [form:3,@mouseover,harm,nodead][form:3] Pummel

Equip Shield Macro
          
#showtooltip
/stopcasting [noworn:Shields]
/equipslot 17 name-of-your-shield
/equipslot 16 name-of-your-main-hand
/cast Defensive Stance
/cast Shield Block

Ranged (all-in-one) Macro
          
#showtooltip
/cast [worn:Thrown] Throw; [worn:Bows] Shoot Bow; [worn:Guns] Shoot Gun; [worn:Crossbows] Shoot Crossbow

LIP Shout Macro
          
#showtooltip Challenging Shout
/use Limited Invulnerability Potion
/cast Bloodrage
/cast Challenging Shout
/s [ AoE Taunted! ]

Mocking Blow Macro
          
#showtooltip Mocking Blow
/startattack
/cast Battle Stance
/cast Mocking Blow

Charge/Intercept Macro
          
#showtooltip [nocombat] Charge; Intercept
/startattack
/cast [nostance:1,nocombat] Battle Stance;[nostance:3,combat] Berserker Stance;
/cast [nocombat] Charge; [combat] Intercept

Cancelling Macro
          
# add cancelaura to a macro you use a lot - like Bloodthirst
/cancelaura Bloodthirst       
/cancelaura Argent Dawn Commission
/cancelaura Arcane Brilliance

# removes error texts such as "not enough rage"
/run UIErrorsFrame:Clear()

Coldrage Dagger Macro
          
#showtooltip Coldrage Dagger 
# double-click to equip both
/castsequence reset=2 Coldrage Dagger
/equipslot 17 Coldrage Dagger
